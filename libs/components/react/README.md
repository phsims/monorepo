# react-components

This library was generated with [Nx](https://nx.dev).

Shared React UI (atoms, molecules, organisms) for apps in this monorepo.

## Feature highlights

- **Feature Image Card** — `components/react/molecules/feature-image-card` (`FeatureImageCard`).
- **Contact Form** — `components/react/organisms/contact-form` (`ContactForm`): name, email, message, Cloudflare Turnstile, POST to your API (see CookbookKeeper `POST /api/contact`).

## Running unit tests

Run `nx test react-components` to execute the unit tests via [Jest](https://jestjs.io).
